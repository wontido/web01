최후의 문법 속성과 img 1월 30일

사진 첨부 태그 < img >
태그 이름만으로는 정보가 부족한 경우 발생
추가 정보를 위하여 속성(Attribute) 추가
이미지 주소 속성: src (source)
< img src="이미지 주소" >
이미지 크기 속성: width
< img src="이미지 주소" width=XX%" >
width: 이미지 크기 지정, 100%, 50%
HTML5에서는 % 대신 pixels 사용으로 숫자만 넣으면 됨
%로 하면 창 크기에 따라 이미지 크기 변함
pixels로 하면 크기 고정
이미지 크기 속성은 height도 있음
alt(Alternate) 속성은 이미지를 로드하거나 로드하지 못했을때 이미지 영역에 글자로 표현해주는 것
