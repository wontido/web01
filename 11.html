부모 자식과 목록 2월 13일

부모 태그와 자식 태그: 부모-자식처럼 연관된 태그들이 있다.
목차 태그 (list)
< li > ___ < / li >
< li > 태그의 부모 태그인 < ul >
경계를 구분하기 위해 필요함
< ul >
< li >1. HTML < / li >
< li >2. XXXX < / li >
< li >3. XXXX < / li >
< / ul >
*만약에 1억개의 목록에서 목록의 숫자를 변경한다면?
< ol > ___ < / ol > 태그를 < ul > 대신 사용한다.
ol: Ordered List
ul: Unordered List

표의 경우 3대가 같이다님
< table >
< tr >
< td > ___ < / td >
< / tr >
< / table >
