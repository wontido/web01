웹사이트 완성 3월 6일

웹페이지를 책처럼 만드는 것이 웹사이트임.
웹페이지를 본드처럼 링크를 이용해 서로 연결해주면 됨.

각 페이지를 저장하는데
웰컴 페이지 -> Index.html
HTML -> 1.html
CSS -> 2.html
Javascript -> 3.html

Head영역의 제목(title)도 각 페이지에 맞게 설정할 것

결국 구조화를 잘 설정하고 연결하는 것이 중요함.
