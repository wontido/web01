HTML이 중요한 이유 1월 30일

1. WEW에서 일반인과 지식인 글쓰기 차이 (편집기)
- 일반인: 글자크기 22 point 진하게 누름 (coding)
< p > < strong > < span style="font-size: 22 px;" > coding < / span > < / strong > < / p >
- 지식인: format에서 Title3 지정
< h3 > coding < / h3 >

→ 검색엔진으로 위의 2가지 비교
웹페이지의 HTML 코드를 분석하여 검색결과 알려줌
제목 코드로 작성된 웹페이지가 검색결과에 우선순위에 나옴

2. WEB은 접근성, 개방성 중요 (accessibility)
시각장애인들은 WEB을 청각화하는 프로그램 사용
만약 이미지로만 WEB을 만들면 접근이 안됨
